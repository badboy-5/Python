# 12306抢票

#### Description
12306抢票小爬虫。本项目基于selenium实现了12306的自动登录抢票，可自动过滑动验证码，自动查询车票是否开售，一旦放票可以自动选票并提交订单，可以选学生票，你只需在10分钟内支付即可。
